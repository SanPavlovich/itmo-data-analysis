**Что делает этот код:**

1. **Загружает данные** из Parquet-файла
2. **Выполняет базовую предобработку:**

- Распаковывает last_hour_activity (если это структура)
- Преобразует булевы колонки в числовые (0/1)
- Удаляет идентификаторы и текстовые поля
- Извлекает час и день недели из timestamp

3. **Готовит данные для ML:**

- Разделяет на train/test (стратифицированно)
- Определяет числовые и категориальные признаки
- Создает пайплайн предобработки

4. **Обучает несколько ML моделей:**
- Logistic Regression
- Random Forest
- Gradient Boosting
- XGBoost
- LightGBM

5. **Оценивает модели:**
- Accuracy, Precision, Recall, F1-Score, ROC-AUC
- Матрицы ошибок
- ROC и Precision-Recall кривые

6. **Сохраняет результаты:**
- Лучшую модель в файл
- Все модели в архив
- Метрики в CSV
- Отчет в текстовый файл

7. **Особенности:**
- НЕ создает дополнительные признаки (кроме базовых из timestamp)
- Использует стратифицированное разделение для сохранения баланса классов
- Поддерживает балансировку классов (SMOTE, ADASYN, undersampling)
- Визуализирует все ключевые метрики
- Генерирует бизнес-отчет с финансовыми оценками
